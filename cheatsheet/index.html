The terminal or git bash

To open on windows, press the windows button and type git-bash. On a mac type cmd + space to open spotlight and type terminal

cd

change directory, moves you into a new directory. To move into the myCode directorycd myCode to move up one directorycd .. ~ is called a tilda, and this command will bring you to your home directorycd ~

ls

lists all the files in this directory, will display a list of all files and directories under this directory ls

git clone

clones a git repository git clone https://github.com/bark4mark/learning-programming.git

to create a function the function keyword is used
